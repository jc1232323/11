## 设计细节

### Header 区域
- 左侧 48px 圆角图标容器（primary-light 背景 + primary 图标色）
- 标题 + 副标题，与化学知识页/设置页的 header 风格一致

### 训练包卡片
- 2 列 grid 布局（`grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))`）
- 每张卡片包含：
  - 顶部：图标圆形徽标 + 专题名称
  - 中部：2-3 行 tag 标签展示覆盖知识点
  - 底部：题目数量 + "开始训练"按钮
- 卡片 hover：`translateY(-2px)` + shadow 加深（复用 `.card-hover`）
- 入场动画：stagger fadeInUp（与化学知识页一致）

### 卡片标签
- 小圆角 pill 样式，`primary-light` 背景 + `primary` 文字
- 表示该专题涵盖的具体考点

### "开始训练"按钮
- `btn btn-primary` 样式，宽度撑满卡片
- 按钮目前只是 UI 展示（不接后端逻辑）

### 配色与动画
- 延续全站 primary 蓝紫色系
- 每张卡片左上角有一个彩色装饰条（4px 宽，primary 渐变色）
- stagger 动画延迟 80ms 递增
